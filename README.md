# App Entrevista
Una app básica por consola que permite encontrar la mejor combinación de elementos para escalar un risco, con un listado de elementos anteriormente dados, basandose en el peso y las calorías que necesite 
el escalador.

## Uso
1. Ejecutar la solución la cual esta dentro de la carpeta AppEntrevista.Presentación con el nombre AppEntrevista.Presentación.sln.
2. Ingresa la cantidad de peso máxima que puede cargar y el número minimo de calorías.
3. Presiona enter para ver el listado de elementos optimos para llevar a la escalada.
4. El resultado se mostrará en la consola.
